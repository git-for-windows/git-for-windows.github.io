Git for Windows: msysGit Community

Git for Windows is a community effort. If you find that things do not work the way you want you can find help on the

msysGit mailing list

or Report Bugs and fix them yourself by using our development environment described below. Note that if you report a bug you must at least be available to provide more details. The fastest way is that you fix it yourself. The community will try to give you a hand by giving some useful pointers...

In order to develop Git for Windows, these volunteers rely on a build environment that is based on the MSys/MinGW project.

Download Git SDK (msysGit)

Audience: Testers, developers, custom installer maintainers

Git repository: Git itself, msysGit Development environment

How to start:

  • msys.bat
  • git-cmd.bat
  • Other, after running share/msysGit/add-shortcut.tcl

Bug tracker: Please ask for help on the msysGit mailing list. If you want to track the bug somehow (e.g. as a todo list for yourself) you can create entries for our git fork and msysGit development. Please note that creating a issue does not mean that it will get fixed. You have to investigate yourself. The community will try to support you with your effort to find a solution. Please have a look at SubmittingPatches and our CodingGuidelines when submitting patches/pull requests.
